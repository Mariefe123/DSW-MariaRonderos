El propósito de este objeto virtual de aprendizaje es compartir una serie de recuros como video, audio, actividades que le permitan a los usuarios recibir conocimientos de una manera divertida y entretenida sobre un tema determinado

Introducciòn

El diseño de sitios web tiene gran importancia para un ingeniero de sistemas, pero lo primero que se debe hacer es diseñar la forma y el estilo que se le quiere dar a una pàgina, ya que esta no solo se desarrolla mediante HTM y CSS, sino que està debe ser agradable para el usuario. Es por eso que en esta fase el estudiante diseña y desarrolla los elementos para construir un sistio web utilizando HTML5 danco estilos con CSS3.
